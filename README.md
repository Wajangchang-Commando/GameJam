## 4. 게임 설명 
  + ### 4 - 1. 조작 방식
      >##### 좌측 버튼을 통해 사용 스킬을 바꾸고, 우측버튼을 눌러 스킬을 사용합니다.
  + ### 4 - 2. 스킬
      >##### 점프 : 잠시 위로 점프 합니다.
      >##### 쉴드 : 방패를 꺼내 공격을 막을 수 있습니다.
      >##### 총알발사 : 총을 들고 발사 할 수 있습니다.
  + ### 4 - 3. 최종목표
      >##### 여러 몬스터를 무찌르고, 최종 보스인 염라대왕을 무찌르는 것이 최종목표입니다.
